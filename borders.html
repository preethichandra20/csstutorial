CSS Borders

CSS Border are used to apply various border-styles for any HTML element. To apply a simple border for any HTML element we use CSS borders.The CSS borders have the following properties.

border-width

This property indicates the width of the border. We can specify the width of a border in px, cm, mm, em, inch etc.
border-width : 5px;

border-style

This property is used to specify the stles of the border. This property contains the following values:-
solid , dashed, dotted,double.
border-style : solid;

border-color

This property is used to specify the color of the border.
border-color : black;

Shortcut-property

We can combine all border-styles using this shortcut property.
Syntax:- border : width style color;
border : 2px solid black;

The above property will apply the border on all the sides. We can even apply the border for individual sides also using the following properties.
border-top : 60px solid pink;
border-right : 80px solid green;

border-radius

We can apply rounded corners for a border of a HTML element by using a CSS property called 'border-radius'
border-radius:20px;
We can even apply border-radius for individual sides also.
border-top-left-radius:20px;
border-bottom-right-radius:30px;
