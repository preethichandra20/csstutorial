CSS Paddings

The CSS paddings indicates some white space inside the border and around the HTML elements.We can specify the paddings in px, %, inherit values.
By using CSS paddings we can generate some white space for individual sides of any HTML element.
padding-top -> to apply the top padding for an element.
padding-right -> to apply the right padding for an element.
padding-bottom -> to apply the bottom padding for an element.
padding-left -> to apply the left padding for an element.

Shorcut Property

By using a shortcut property we can generate some padding for individual sides using a CSS property called 'padding'.

  1. The padding property values starts from top to left in clockwise manner.
    syntax:- padding : Top Right Bottom Left;
  2. If we provide two values for padding property then the first value indicates top and bottom padding, and the second values indicates the left and right side of a padding.
    syntax:- padding : Top-Bottom Left-Right;
  3. If we provide only one value,this indicates the padding of all sides.
    syntax:- padding : All-Sides;
