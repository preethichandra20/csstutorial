Internal Approach

In this approach we will apply the CSS styles using HTML<style>tag. We have to add <style> tag in the head section of the HTML document.

To apply the text color red for all the paragraphs of the HTML documents is as follows
<style>
p{
color:red;
}
</style>



Drawbacks

  1. In this approach we can apply styles only for the specific HTML document,these styles will not be applied to any other HTML document
  2. This approach increases the code duplication and code redundancy.
  3. This approach increases the maintainability of the application.

In order to the resolve this issue External approach is preferred.
