CSS Colors

To specify a color for any HTML element we use various color values.But for each color many numbers of shades are possible.In order to specify all shades using normal text color is highly impossible.Using CSS color values we can specify all the shades of a color. CSS colors contain the following values.

Textual Color

In this we will specify the color values using textual format.Using this way we can use a few color values only We cannot be able to specify the exact shade of the required color.
background-color:green;

RGB

In this approach we will use various combinations of color values of red,green,blue.
Here the color value starts from 0-255.
background-color : rgb(0,0,0); //black color
background-color : rgb(255,255,255); //white color

HEX(Hexa Decimal)

In this approach we will specify the color values using hexadecimal format of color values with the combination of red,green,blue.
Here the color values starts from 00-FF.
background-color : #000000; //black color
color : #FFFFFF; //white color

HSL(Hue,Saturation,Lightness)

This is another way of providing colors.
Hue -> A degree on the color wheel. (0:red,120:green,240:blue,360:black).
Saturation -> A percentage value. (0% means a shade of gray, and 100% is full color)
Lightness -> A percentage value. (0% is black, 50% is neither light nor dark, 100% is white).
#hsl{
background-color : hsl(220,45%,75%);
}

RGBA(red,green,blua,alpha)

This is same as RGB with an extra parameter alpha.Here 'alpha' indicates the transparent value,its value between 0&1.
0 -> full transparent; 1 -> full color
#rgba{
background-color : rgba(66,103,178,0.5);
}

HSLA(Hue,Saturation,lightness,alpha

This is exactly same as HSL with an extra parameter 'alpha' Here 'alpha' indicates the transparent value,its value between 0&1.
0 -> full transparent; 1 -> full color
#rhsla{
background-color : hsla(220,45%,47%,0.5);
}
