External Approach

This is the best approach to apply styles to any webpage.In this approach we will create a separate '.css' file and apply styles to it. We can connect this '.css' file to any HTML document using Html <link> tag.

Ex:
<link rel="stylesheet" href="style.css">
We have to add this link to the head section of all the Html documents of a website.
