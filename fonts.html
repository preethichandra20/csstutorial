CSS Fonts

The CSS fonts are used to applu the font styles for HTML content like font-family, font-size, font-weight and font-styles
The CSS Fonts contains the following properties.

font-family

This property is used to apply the type of the font to be applies for the HTML content.
In CSS all the fonts are categorized into four families, such as

  1. serif (Times New Roman)
  2. sans-serif (Arial, Verdana ,Georgia)
  3. monospace (Lucida Console)
  4. cursive (most of the google fonts)

#arial{
font-family : Arial,sans-serif;
}



font-style

This property is used to covert normal text to italic.
#font{
font-style : italic;
}

font-weight

THis property is used to covert the normal text to bold or bolder
#font{
font-weight : bold;
}

font-size

This is used to apply the custom font size for the HTML element in px, cm, mm etc.
#font{
font-size:20px;
}
