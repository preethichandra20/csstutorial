CSS Links

CSS Links are used to apply the custom styles for hyperlinks for some special events using CSS pseudo selectors.
Usually each HTML hyper links are having the default syles link un-touched links with blue in color and underline and visited links will be in blue color.
We can customize these styles. The CSS linkes pseudo selectors are as follows:-

a:link

This pseudo selector indicates a normal unvisited and un-touched block.
a:link{
background-color:forestgreen;
}

a:hover

This pseudo selector indicates the moment a user mouse hover on a hyperlink.
a:hover{
background-color:orangered;
}

a:active

This pseudo selector indicates the moment a user mouse click but not released the cursor on the link.
a:active{
background-color:black;
}

a:visited

This pseudo selector indicates the link which has been visited already.
a:hover{
background-color:blue;
}
