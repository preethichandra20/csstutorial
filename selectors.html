CSS Selectors

We normally use CSS to select a specific HTML elements and apply styles to it.To selecr any HTML element we use the following CSS selectors

The CSS selectors are classified as follows

  1. Element Selector/Tag Selector
  2. Id selector
  3. Class Selector
  4. Attribute Selector
  5. Pseudo Selector

Element Selector

In this selector we select a specific element or tag and apply styles to it.
To apply the background-color red for a specific tag or element is as follows.
p{
background-color : red;
}

Id selector

In order to apply some styles for a specific element,we have to give a unique id and we select this id and apply styles to it.
#green{
background-color:green;
}
<p id="green>Lorem ipsum dolor sit amet.</p>

Class Selector

In the class selector we can add the same class name for a group of elements and apply styles to it.
To apply the background-color green for a group of HTML elements using class selector as follows,
.green{
background-color:green;
color:white;
}
<p class="green>Lorem ipsum dolor sit amet.</p>
<h3 class="green>Lorem ipsum dolor sit amet.</h3>

Attribute Selector

By using CSS we can even select a specific attribute of a HTML element and apply styles to it.These styles are used when we apply some styles only for a text box or password box.
To apply background-color red fir a password box using attribute selector as follows,
imput[type="password"]{
background-color:red;
}

Pseudo Selector

An action which a user is performing on the webpage is called as an event.This is the selector to apply CSS styles for special events on the browser.
To apply the background color red which focus on a text box is as follows,
imput[type="password"]:focus{
background-color:red;
}
