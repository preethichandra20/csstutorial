Inline Approach

In this process of applying styles to HTML elements we use 'style' attribute.


Drawbacks

  1. This approach is only used to apply styles for a specific tag only.
  2. This approach increases the code redundancy and code duplication.
  3. This increases code maintainabilty of the application.

In order to resolve this issue,internal approach is preferred.
