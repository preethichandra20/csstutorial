CSS Backgrounds

The CSS backgrounds properties are used to apply various background styles for the HTML elements.The CSS Backgrounds has the following properties.

background-color

This property is used to apply the any background-color for the HTML element using textual color ,RGB, HEX, HSL, RGBA, HSLA
background-color : rgba(0,0,0);

background-image

This property is used to apply the background-image for any HTML element
background-image: url("img/image5.jpeg");

background-repeat

This property is used whether to repeat the image in the background or non-repeat. This property contains the following values,
repeat-x -> to repeat image in X-axis.
repeat-y -> to repeat image in Y-azis.
no-repeat -> to display a single image.
background-repeat : no-repeat;

background-attachment

This property is used to fix the image in the background or scroll the image along with the scroll bar.This property contains the following values.
fixed -> to fix the image in the background.
scroll -> to scroll the image along with the scroll bar.
background-attachment : fixed;

background-position

This property is used to place the image in the specified position like right, left, bottom, top, center
background-position : right;

Shortcut property for background styles

We can apply all background properties using a single CSS property called 'background'
syntax :- background : color url repeat attachment position;
background : white url("img.ab.jpg") no-repeat fixed bottom;
