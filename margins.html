CSS Margins

The CSS Margins indicates some space around the border of any HTML element. Any white space which is around the border of any HTMl elemnt comes under margin.
We can apply the individual sides of margin using the following CSS margin properties.
margin-top -> to apply the top margin for an element.
margin-right -> to apply the right margin for an element.
margin-bottom -> to apply the bottom margin for an element.
margin-left -> to apply the left margin for an element.
we can specify the margin in px, %, inherit values.

Shorcut Property

By using a shortcut property we can generate some margin for individual sides using a CSS porperty called 'margin'.

  1. The margin property values starts from top to left in clockwise manner.
    syntax:- margin : Top Right Bottom Left;
  2. If we provide two values for a margin property then the first value indicates top and bottom margin, and the second values indicates the left and right side of a margin.
    syntax:- margin : Top-Bottom Left-Right;
  3. If we provide only one value,this indicates the margin of all sides.
    syntax:- margin : All-Sides;
